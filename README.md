# 📝 Simple To-Do List App

A clean and functional To-Do List application built to manage daily tasks efficiently. This project focuses on DOM manipulation and local storage to ensure tasks persist even after refreshing the browser.

## ✨ Features
* **Add Tasks:** Quickly add new items to your list.
* **Mark as Done:** Toggle completion status with a single click.
* **Delete Tasks:** Remove tasks you no longer need.
* **Responsive Design:** Works smoothly on mobile, tablet, and desktop screens.
* **Data Persistence:** Uses Browser LocalStorage so your data isn't lost on refresh.

## 🛠️ Tech Stack
* **HTML5:** For semantic web structure.
* **CSS3:** For custom styling and responsive layout.
* **JavaScript (Vanilla):** For dynamic task management and logic.

## 📂 Project Structure
* `index.html` - The main entry point.
* `style.css` - Custom styles and animations.
* `script.js` - The logic for adding, deleting, and storing tasks.

## 📖 How to Use
1. Clone the repository or download the ZIP.
2. Open `index.html` in any modern web browser.
3. Start adding your tasks!
